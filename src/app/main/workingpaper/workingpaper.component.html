Quarantine affected machines

Business Unit
Allen Avenue Branch
Raised By:
John AKindele

Creation Date:
10.07.2014 23:36:57

Risk Details:

Virus and malware can easily transmit to other computers via the network either WAN or LAN
Severity:

Medium


Control:

Isolate all affected computers from the network and ensure they all pass Anti-Virus scan before placing back on the network


sample Size:

23

Attritbutes

Information Decision
Can the affected computer still be fixed
Where storages on the network affected
Does it require physical device components to be changed
Is there a cost attached to resolution
